# Install Dependencies

1. 'python 2.7'
	https://www.python.org/downloads/
2. `cv2`
	https://gist.github.com/arthurbeggs/06df46af94af7f261513934e56103b30
3. `sklearn`
	pip install sklearn
4. `skimage`
	pip install scikit-image
5. `numpy`
	pip install numpy
6. `collections`
	pip install collections-extended

# How to Run
